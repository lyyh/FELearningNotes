## 题目
```
['1','7','11'].map(parseInt) 返回 [1,NaN,3]
```

## 解答
对于'1'转换的时候等同于  
```
parseInt('1',0,['1','7','11']) // 忽略第三个参数，第二个参数为 false 时，默认是10进制转化
```
对于'7'转换的时候等同于  
```
parseInt('7',1,['1','7','11']) // 忽略第三个参数，'7'符号不能作为一进制表示，所以为NaN
```
对于'11'转换的时候等同于  
```
parseInt('11',2,['1','7','11']) // 忽略第三个参数，'11'符号作为二进制表示成3
```
